# AEM [!DNL Forms] versões {#aem-forms-releases}

Adobe Experience Manager [!DNL Forms] é um aplicativo implantado no AEM. Ele estende experiências envolventes em Web e dispositivos móveis a formulários e documentos corporativos, permitindo que as organizações completem interações digitais complexas e, ao mesmo tempo, melhorem as experiências dos usuários e estendam o alcance dos negócios. A AEM [!DNL Forms] incluída AEM start rápido tem um conjunto limitado de funcionalidades. AEM [!DNL Forms] pacote complementar fornece recursos avançados, como integração com [!DNL Adobe Analytics], [!DNL Adobe Sign], serviços de documentos, motor de workflow e muito mais.

>[!NOTE]
>
>A AEM Forms lança os pacotes complementares uma semana após a data programada de lançamento do AEM Service Pack e do Cumulative Fix Pack.

A lista a seguir contém todos os pacotes complementares AEM [!DNL Forms] lançados até hoje, versões AEM correspondentes (Pré-requisitos), links de download de pacotes e outras informações úteis:

## Coisas para lembrar {#things-to-remember}

* Os pacotes de complementos AEM Forms listados acima são aplicáveis a AEM [!DNL Forms] no OSGi e AEM [!DNL Forms] no JEE.
* Todos os pacotes complementares AEM [!DNL Forms] e as Correções rápidas são cumulativos; ou seja, um pacote de complementos e o Quick Fix incluem todos os complementos e Correções rápidas anteriores para um componente da versão correspondente. Por exemplo, se você estiver instalando o Quick Fix 2.4.50, ele já incluirá AEM [!DNL Forms] pacote complementar 2.4.26.

   >[!NOTE]
   >
   >Os pacotes de complementos cumulativos e as Correções rápidas implicam que os dois primeiros números da versão sejam idênticos.

* Sempre instale o complemento disponível mais recente do Service Pack/Feature Pack. É altamente recomendável, pois inclui correções e melhorias de segurança, desempenho, estabilidade e essenciais para o cliente lançadas desde a disponibilidade geral da versão base.
* Sempre que você instalar um Service Pack ou atualizar de uma versão anterior do AEM para uma versão mais recente, instale o pacote adicional AEM [!DNL Forms] correspondente. Por exemplo, se você atualizar do AEM 6.1 [!DNL Forms] add-on package versão 2.2.4 para AEM 6.1 Service Pack 1, após instalar o AEM 6.1 Service Pack 1, instale AEM [!DNL Forms] add-on package versão 2.4.50.
* Não há suporte para a desinstalação do pacote complementar AEM [!DNL Forms].
